第四篇 选择托管网站的方式以及系统 | 从零创建一个属于自己的网站(小白向)

发表于2023-08-12|更新于2023-08-12|教程Web
|总字数:1.5k|阅读时长:4分钟

虚拟主机 Or VPS?

网站新手一般我建议托管，而不是本地部署后进行端口映射之类的

托管的话会比自己本地部署省心很多（前提是选择正确的厂商），虽然长期来看会花费更多的钱财，但是对于新手来说采用托管至少可以省去很多系统安装的麻烦

当然后期自己熟稔起来之后也可以尝试在本地部署，会省钱很多的

虚拟主机

虚拟主机其实就和我的世界面板服一样，你的所有操作都在云服务商给你的操作面板上进行操作，所以限制很多

但好处也是有的，就是价格便宜（相对的），使用方便，不用自己操心环境之类的

不过虚拟主机也就仅限于搭建网站了，玩不了更多有趣的东西，并且没有属于自己的公网IP

还有一点，有些虚拟主机不支持改环境，所以有些对环境有要求的网站是无法搭建的，例如需要额外的phh扩展的网站

VPS

VPS的可玩性就大大增强了，因为VPS就相当于你有了一台拥有公网ip的电脑（至于配置当然是自己选，看自己财力），你可以远程操控它

你可以在上面做任何你喜欢的事情（违反国家法律，服务商用户守则的事情除外），你可以搭建除网站以外更多有趣的服务，像alist，cloudreve之类的

甚至是我的世界服务器（前提是有马内，不过1h2g的服务器实测1.16.5以前的纯净服几个人联机一起玩还是没有问题）

但是它也有一个缺点，就是有一点贵，不过现在腾讯和阿里等几个大厂推出了轻量云服务器，价格会好很多

如果只是个人搭建网站玩的话完全够用了

serverless

这是一种较新的网站托管方式

找了一个比较官话的介绍https://www.zhihu.com/tardis/zm/art/78250791?source_id=1005

用我的话来说，就是=白嫖（doge

但是不是什么网站程序都可以的，它需要网站程序能够适配，所以我一般也就用来托管静态网页或者部分已经适配serverless的服务

例如经典的开源网站流量统计umami就支持serverless，目前我在用的在国内比较好用的免费的serverless托管服务商就是vercel，其他的可以自行去百度

Windows Or Linux

这个其实要根据你的服务，服务器的配置，服务器的用途等等来决定

Windows

好处就是可视化，有着和家用Windows10差不多的界面，使用起来非常亲切

真的就和远程玩电脑一样的感觉，你甚至可以用来追剧玩游戏（看你财力）

可以运行一些Windows平台独有的服务，可以24小时挂机百度网盘（doge

有IIS（Windows平台独有），可以运行aspx等网站文件

缺点就是占用大（Windows吃硬件更多），很多Linux的程序或者服务无法搭建使用

开docker的话因为有一层Linux的虚拟机所以会导致额外的性能开销

Linux

系统开销小，可以直接安装docker，可以学习更多知识

and听说在Linux上装MySQL数据库比在Windows上装性能更好？这个不知道是不是真的

开源免费

缺点就是没有可视化界面（当然你可以自己装个桌面，也能实现可视化，但其实没有必要），不能追剧，不能运行Windows平台独有的程序，不能使用aspx等网站程序（不过现在一般都用的是php了吧）

没装桌面的时候就是简单的命令行，上手有一定难度，没有防呆设计，rm -rf /* 即可删库跑路，带走一切

我的建议其实还是使用Linux比较好，因为你始终是要接触它的。在内存是金子的服务器这里，能节约一点内存是一点（doge

面板

如果你听从了我的建议选择了Linux，那么这里有一款面板我极力推荐—>1panel

什么，你不知道什么是面板？面板就和虚拟主机一样的，通过web面板来管理VPS，但是他比虚拟主机的面板功能丰富多了，实现真正管理自己的VPS

什么，你不知道什么是1panel？

1Panel 是一个现代化、开源的 Linux 服务器运维管理面板。

  • 快速建站：深度集成 Wordpress 和 Halo，域名绑定、SSL 证书配置等一键搞定；
  • 高效管理：通过 Web 端轻松管理 Linux 服务器，包括应用管理、主机监控、文件管理、数据库管理、容器管理等；
  • 安全可靠：最小漏洞暴露面，提供防火墙和安全审计等功能；
  • 一键备份：支持一键备份和恢复，备份数据云端存储，永不丢失。

这是1panel的官网1Panel - 现代化、开源的 Linux 服务器运维管理面板

所以这就是为啥我推荐Linux，有了管理面板，根本就不用怕那个黑黢黢的命令窗口，一切又变成可视化操作了，而且系统占用的资源还少，这不嘎嘎爽？

后记

如果你的资金还勉强过得去的话，还是建议上VPS，我记得阿里云的香港轻量最低一个月才24块钱（前提是有货，太抢手了），可玩性真的很高

不要问我为啥不推荐宝塔，我就是从宝塔转到1panel的，问就是不喜欢答辩宝塔一堆收费的功能，关键是有些功能还简单的要命

对了，如果你选择了Linux，记得学习Linux的相关知识，这里不教的（doge
